# Mihits 项目介绍

Mihits 是一套 **C/S 架构的在线游戏大厅**：Qt 桌面客户端 + Linux 大厅服务端（epoll）。用户登录后进入统一大厅，可选择内置五子棋对战，或拉取并启动外部独立游戏（如音游 Malody-Martix）。服务端负责账号、专区/房间、实时转发与录像；客户端负责界面、棋盘对局与外部游戏的版本校验、增量更新及进程拉起。

---

## 项目功能

### 账号与大厅
- 注册 / 登录（服务端校验，登录下发重连 token）
- 主界面专区入口（五子棋、Malody 等）
- 心跳保活与断线提示 / 重连

### 五子棋在线对战
- 加入专区、查看房间人数、进出房间
- 房主 / 玩家 / 观战身份与准备、开局
- 落子同步、胜负判定与录像存档 / 回放
- 托管（AI）与观战 AI 提示

### 断线与房间通知
- TCP 软断线：保留会话，房间内对手置灰
- 心跳超时硬断线：清房间状态并通知对手
- 软断窗口内重连：恢复房间成员与对局棋谱重放

### 外部游戏专区（Malody-Martix）
- 点击专区后校验版本（大厅 TCP）
- 按 MD5 清单增量更新（HTTP / Nginx），多余文件清理
- 更新成功后加入专区、隐藏大厅、拉起独立进程
- 游戏退出后自动返回大厅并离开专区

### 服务端能力
- epoll + 线程池处理连接与业务
- MySQL 持久化账号、录像、外部游戏包元数据
- 房间广播与对局落子缓存（支持重连重放）
